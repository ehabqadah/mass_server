File Upload Script

The file upload script can be used to submit malware samples to the MASS server directly from the command line. The script supports submission of single files or a complete directory. You can optionally set a comment which will be attached to the uploaded samples.

Usage: 
./mass_cli_upload.sh -u <api_url> (-d <directory>|-f <file>) [-c <comment>]
